# HDIC Samples Agent Guide

## General Principles

- TSV files are UTF-8
- Preserve original orthography
- Do not normalize variant characters automatically
- Katakana wakun are important linguistic data
- Historical annotations should be preserved

## Recommended Tools

Use:
- pandas
- matplotlib
- regex
- pathlib

Avoid unnecessary frameworks.

## Preferred Analyses

- frequency analysis
- dictionary comparison
- wakun analysis
- definition pattern extraction
